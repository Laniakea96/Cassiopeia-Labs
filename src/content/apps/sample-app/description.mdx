## About Sample App

This is a placeholder app to demonstrate how the Cassiopeia Labs website works.

When you're ready to add your real apps, simply:

1. Edit `src/data/apps.ts` to add your app's metadata
2. Create a folder in `src/content/apps/your-app-slug/` with MDX files
3. Add your app's icon and screenshots to `public/images/apps/your-app-slug/`
